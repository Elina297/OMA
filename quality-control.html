top_features <- getTopTaxa(se, method="median", top=10)
tax_data <- rowData(se)[top_features,taxonomyRanks(se)]

4.2 Library size

The total counts/sample can be calculated using the perCellQCMetrics/addPerCellQC from the scater package. The first one just calculates the values, whereas the latter one directly adds them to the colData.
